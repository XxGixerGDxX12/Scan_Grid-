# Scan_Grid-

This simple script is a network scanner that uses nmap 

# Before Installation: 

```
apt update && apt upgrade
```
```
pkg install nmap python
```

# Install Termux:

```
git clone https://github.com/XxGixerGDxX12/Scan_Grid-/
```

```
cd Scan_Grid
```

```
python3 net_scan.py --router (Your Router IP)
```
